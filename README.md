# PKU-AFMU

## Quickstart

> require `python3.7`

```shell
$ pip3 install virtualenv

$ virtualenv venv --python=python3.7

$ source ./venv/bin/activate

$ pip3 install -r requirements.txt

$ ./dev_start.sh    # run locally
$ ./dev_start.sh -g # or run with gunicorn
```
